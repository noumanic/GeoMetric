Assignment 2: Multi-Map Spatial Analytics 
in Python 
Title 
From Raw Spatial Data to Insightful Maps: A Python-Based Geospatial Visualization 
Assignment 
Objective 
You are hired as a junior geospatial data analyst. Your task is to build a small map analytics 
portfolio in Python using multiple real-world datasets. You must solve different spatial problems 
using different map types, justify your design decisions, and reflect on the strengths and 
weaknesses of each visualization. 
This assignment is not only about plotting maps. It is about answering the question: 
When is a map necessary, which map type is appropriate, and what distortions or misleading 
effects must be considered? 
Learning Outcomes 
By the end of this assignment, you should be able to: 
1. Load, clean, merge, and visualize geospatial datasets in Python. 
2. Choose appropriate map types for different spatial questions. 
3. Explain when a choropleth, proportional symbol map, contour map, flow map, or 
cartogram is appropriate. 
4. Use and compare projections. 
5. Avoid misleading geographic representations. 
6. Build reproducible geospatial analysis workflows.

Part 1 – Projection Challenge 
Problem 
Create the same world or national dataset using at least three different projections and 
analyze how interpretation changes. 
Required projections 
Choose at least 3 from: 
• 
Mercator 
• 
Equirectangular 
• 
Azimuthal 

• 
Albers Equal-Area 
• 
Lambert Conformal Conic 
• 
Winkel Tripel or Robinson if available 
Tasks 
1. Plot the same dataset in 3 projections. 
2. Explain what is preserved and what is distorted. 
3. Identify which projection is most appropriate for: 
o navigation 
o thematic comparison by area 
o general educational world map 
Example datasets 
• 
world population by country 
• 
greenhouse gas emissions 
• 
country area vs population 
Expected insight 
Students should show that projection is not a cosmetic choice; it changes perception of area, 
distance, and direction. 
Output 
• 
3 maps 
• 
1 comparison table 
• 
1 short discussion 
 
Part 2 – Choropleth Map and Its Pitfalls 
Problem 
Build a choropleth map for a regional variable and then critique its limitations. 
Example questions 
• 
Which regions have highest disease rates? 
• 
Which counties have highest unemployment? 
• 
Which districts have highest literacy? 
• 
Which countries have highest child mortality? 

Tasks 
1. Create a choropleth map using normalized data. 
2. Use an appropriate classification method: 
o quantiles 
o equal interval 
o natural breaks 
3. Compare at least two classification strategies. 
4. Explain when the choropleth is misleading. 
5. Discuss why absolute counts should usually not be mapped directly on choropleths. 
Required analysis 
Students must explicitly answer: 
• 
Why should ratio/rate data be preferred over raw counts? 
• 
How does region size affect interpretation? 
• 
What visual bias appears when large but sparsely populated regions dominate? 
Output 
• 
2 choropleths 
• 
classification comparison 
• 
short critique 
 
Part 3 – Proportional Symbol Map 
Problem 
Visualize point-based or aggregated magnitude data using proportional symbols. 
Example questions 
• 
Which cities have the largest populations? 
• 
Where are earthquake magnitudes highest? 
• 
Which hospitals have the highest patient loads? 
• 
Which airports have the highest traffic? 
Tasks 
1. Plot symbol locations correctly. 
2. Scale symbol size proportionally to magnitude. 

3. Explain whether symbol area or symbol radius was used. 
4. Reduce overlap using transparency, jitter, zoomed inset, or interactivity. 
5. Compare the proportional symbol map with a choropleth for the same variable where 
possible. 
Required conceptual point 
Students must explain why proportional symbol maps are often better than choropleths for point 
phenomena or unevenly distributed data. 
Output 
• 
1 static map 
• 
1 interactive map 
• 
short comparison paragraph
